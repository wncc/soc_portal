📋 Bulk Import Projects into {{ domain.name }}

Instructions

Method 1: Upload File (Easiest)

  1. Prepare your spreadsheet in Excel/Google Sheets with the required columns
  2. Save/Export as CSV or TSV file
  3. Click "Choose File" and select your file
  4. Click "Import Projects"

Method 2: Copy & Paste

  1. Prepare your spreadsheet in Excel/Google Sheets
  2. Select all data including headers (Ctrl+A)
  3. Copy (Ctrl+C / Cmd+C)
  4. Paste into the text area below
  5. Click "Import Projects"

Required Columns:

  • title - Project title (required)
  • mentee_max - Maximum number of mentees (required)
  • mentor - Main mentor in format "Name (roll)" or just "roll" (required)

Optional Columns:

  • general_category - Machine Learning, Development, Blockchain, etc.
  • specific_category - More specific category
  • co_mentors - Format: "Name1 (roll1), Name2 (roll2)"
  • weekly_meets - Number of weekly meetings
  • description - Project description
  • timeline - Project timeline
  • checkpoints - Project checkpoints
  • prerequisites - Prerequisites for the project
  • banner_image_link - URL to banner image

📝 Spreadsheet Template

Download template files from:

  • socbackend/projects/BULK_IMPORT_TEMPLATE.csv - CSV format
  • socbackend/projects/BULK_IMPORT_TEMPLATE.tsv - TSV format

Or copy this header row to your spreadsheet:

title	general_category	specific_category	mentee_max	mentor	co_mentors	weekly_meets	description	timeline	checkpoints	prerequisites	banner_image_link
